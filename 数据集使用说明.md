# 数据集使用说明

## 一、本项目数据集说明

本项目（方向3：专升本知识点智能答疑助手）的核心数据为**自建的机器学习知识库**，并参考课程文档推荐的公开问答数据资源构建。

### 1. 自建机器学习知识库（主数据集）

| 项目 | 说明 |
|------|------|
| 数据集名称 | 机器学习专升本核心知识点库 |
| 文件 | `data/knowledge_base.csv` |
| 规模 | 105 个知识点（满足"≥100 个核心知识点"要求） |
| 字段 | `id, category, title, content, keywords` |
| 类别 | 监督学习、无监督学习、模型评估、特征工程、神经网络、数学基础（6 类） |
| 构建方式 | 依据《机器学习基础及应用》课程项目 1-11 知识体系，由 `src/build_kb.py` 结构化生成并校验 |
| 用途 | ① 朴素贝叶斯知识点分类训练；② 两级检索答疑；③ 模拟试卷自动出题 |

> 知识点内容覆盖：线性/逻辑回归、决策树、随机森林、SVM、KNN、朴素贝叶斯、K-means、层次聚类、DBSCAN、PCA、过拟合/欠拟合、偏差方差、正则化、梯度下降、交叉验证、各类评估指标（准确率/精确率/召回率/F1/AUC/ROC/混淆矩阵）、标准化/归一化、独热编码、激活函数、反向传播、CNN/RNN/LSTM、集成学习、RAG 等。

## 二、数据预处理说明

`src/build_kb.py` 在生成知识库时完成以下预处理与校验（对应"数据获取与预处理"评分项）：

1. **缺失值检查**：确保所有字段无空值。
2. **重复值检查**：知识点标题去重，避免冗余。
3. **类别合法性检查**：所有类别须属于预定义的 6 类。
4. **特征工程**：文本经 jieba 中文分词 + 停用词过滤 + TF-IDF（1-2 gram）向量化；标题加权拼接以提升区分度。
