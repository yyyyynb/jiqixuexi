# 专升本机器学习知识点智能答疑助手 —— 项目文档

> 《机器学习基础及应用》课程期末大作业 · 方向3：教育服务类 · **进阶版**

---

## 一、需求分析

### 1.1 业务背景

专升本考生在备考"机器学习"科目时，面临知识点零散、缺乏即时答疑、复习缺少针对性等痛点。传统答疑依赖人工，效率低、覆盖面有限。随着大模型与 RAG 技术成熟，构建一个能**自动答疑、文档问答、模拟测验、薄弱项分析**的智能体，可显著提升备考效率。

### 1.2 项目目标

开发一个面向机器学习专升本考试的 AI 智能答疑助手，实现：

- **基础版（必做）**
  1. 整理 ≥100 个核心知识点，构建结构化知识库；
  2. 使用朴素贝叶斯实现知识点分类与检索；
  3. 搭建智能体，支持自然语言提问并返回对应知识点解答。
- **进阶版（选做）**
  1. 集成 RAG，实现 PDF 文档上传与自动问答；
  2. 错题收集与知识点薄弱项分析；
  3. 模拟试卷自动生成与评分。

## 二、技术路线

### 2.1 整体架构

```
                  ┌─────────────────────────────────────────┐
                  │        Streamlit 智能体（app.py）          │
                  │ 答疑 | PDF问答 | 模拟考试 | 错题 | 评估 |
                  └───────┬─────────┬──────────┬──────────────┘
                          │         │          │
              ┌───────────▼──┐  ┌───▼────┐  ┌──▼───────┐
              │ 两级检索      │  │  RAG    │  │ 出题/评分 │
              │ retriever.py │  │ rag.py  │  │ exam.py  │
              └──────┬───────┘  └────┬────┘  └────┬─────┘
                     │               │            │
          ┌──────────▼───────┐       │            │
          │ 朴素贝叶斯分类器   │       │            │
          │ train_classify.py│       │            │
          └──────────────────┘       │            │
                     │               │            │
          ┌──────────▼───────┐       │            │
          │ 知识库 105 条      │       │            │
          │ build_kb.py      │       │            │
          └──────────────────┘       │            │
```

### 2.2 核心技术点

1. **中文文本处理**：jieba 分词 + 停用词过滤 + TF-IDF（1~2 gram）向量化。
2. **知识点分类**：朴素贝叶斯（主）、决策树、随机森林，GridSearchCV 网格搜索调参。
3. **两级检索**：先用朴素贝叶斯预测问题类别缩小范围，再在 TF-IDF 空间用余弦相似度排序，类内优先返回，提升检索准确率。
4. **RAG 检索增强问答**：PDF→pdfplumber 解析→带重叠分块→TF-IDF 向量化→余弦检索 Top-K；返回检索依据。
5. **降级容错**：pdfplumber 缺失时自动切换纯检索/模板模式，保证离线可演示。

## 三、实验结果

### 3.1 知识库

- 知识点总数：**105 条**（满足 ≥100 要求）
- 类别分布：监督学习 21、模型评估 18、数学基础 18、神经网络 17、特征工程 17、无监督学习 14
- 校验：无缺失值、无重复标题、类别合法 ✅

### 3.2 三种分类算法对比（测试集 25%，分层划分）

| 模型 | 准确率 | 精确率(macro) | 召回率(macro) | F1(macro) | AUC(ovr) | 最优参数 |
|------|-------|--------------|--------------|-----------|----------|---------|
| **朴素贝叶斯** | **0.741** | **0.794** | **0.742** | **0.757** | **0.917** | alpha=0.1 |
| 随机森林 | 0.630 | 0.641 | 0.625 | 0.610 | 0.865 | n_estimators=100, max_depth=20 |
| 决策树 | 0.407 | 0.423 | 0.417 | 0.412 | 0.643 | max_depth=20, min_samples_split=3 |

**结论**：朴素贝叶斯在本文本分类任务上全面领先（F1=0.757，AUC=0.917），符合其在高维稀疏文本场景下的优势，故选为知识点分类与两级检索的主模型。

> 混淆矩阵与 ROC 曲线见 `models/confusion_*.png`、`models/roc_*.png`，并在智能体"模型评估看板"标签页可视化展示。

### 3.3 检索效果示例

输入"什么是过拟合，怎么解决" → 预测类别"模型评估" → Top-1 命中"过拟合"知识点（相似度 0.36），Top-2/3 命中相关的"欠拟合""学习曲线"，类内检索准确。

### 3.4 智能体功能

五大标签页均正常运行：智能答疑、PDF 文档问答（RAG）、模拟考试、错题/薄弱项分析、模型评估看板。Streamlit 应用启动正常（本地 HTTP 200）。

## 四、结论与展望

### 4.1 结论

本项目完整实现了方向3的基础版与进阶版全部要求：构建了 105 个知识点的结构化知识库；实现并对比了 3 种分类算法（朴素贝叶斯最优）；通过两级检索与 RAG 提供有依据的知识答疑；同时完成错题薄弱项分析与模拟试卷自动生成评分，最终集成为一个可演示的 Streamlit 智能体。系统具备良好的降级容错能力，离线环境下仍可演示核心功能。

### 4.2 展望

1. **向量检索升级**：用 BGE/通义 Embedding 替代 TF-IDF，提升语义检索准确率。
2. **知识库扩充**：接入 Stack Exchange、维基百科语料，扩展到更多科目。
3. **多轮对话**：增加对话记忆，支持追问与上下文澄清。
4. **部署上线**：将智能体部署为公开 Web 应用或迁移到 Coze/Dify 平台。
5. **自动组卷增强**：按知识点难度与薄弱项个性化组卷，形成"测验—分析—复习"闭环。
